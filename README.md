# Vue Unsplash

## About

The developer learned Vue and created this app in several hours. This is a simple image search app.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
